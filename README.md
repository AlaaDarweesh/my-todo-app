AR: تطبيق "إدارة المهام الذكية" هو مشروع متكامل (Full-Stack) مصمم لزيادة الإنتاجية. يعرض هذا المشروع التكامل القوي بين واجهة خلفية عالية الأداء مبنية بـ NestJS وواجهة أمامية تفاعلية بـ React، مما يوفر تجربة مستخدم سلسة مع تحديثات فورية للبيانات وتواصل فعال عبر RESTful API.

 Key Features | المميزات الرئيسية
Full CRUD Operations: Seamlessly Create, Read, Update, and Delete tasks. (إدارة كاملة للمهام: إضافة، عرض، تعديل، وحذف).

State Persistence: Real-time data fetching and UI synchronization. (تزامن فوري بين البيانات وواجهة المستخدم).

Smart Indicators: Active task counter and completion status tracking. (عداد ذكي للمهام النشطة وتتبع حالة الإنجاز).

Clean Architecture: Separated concerns between Frontend and Backend logic. (معمارية برمجية نظيفة تفصل بين منطق الواجهة والسيرفر).

Modern UI: Fully centered, responsive, and user-friendly design. (واجهة عصرية، موسطة بالكامل، وسهلة الاستخدام).

 Tech Stack | الوسائل التقنية
Frontend: React.js, TypeScript, CSS3 (Flexbox/Grid).

Backend: NestJS (Node.js framework), TypeScript.

Communication: RESTful API (HTTP Methods: GET, POST, PATCH, DELETE).

 How to Explore | كيف تطلع على المشروع
Backend: Explore the /server folder to see the NestJS controllers and services.

Frontend: Explore the /client folder for the React components and state management.
